- Each activity is the closed interval `[si, fi]`
- A set S of intervals is **conflicting** iff every pair of intervals in S overlaps
	- The minimum number of conflicting sets is 4
- Activity `i` **overlaps** with `j` if they share a common point i.e. `si ≤ t ≤ fi` & `sj ≤ t ≤ fj`
