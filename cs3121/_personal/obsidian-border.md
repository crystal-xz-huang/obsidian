# Obsidian Border Theme

[GitHub - Akifyss/obsidian-border: A theme for obsidian.md](https://github.com/Akifyss/obsidian-border#obsidian-border-theme)

## Style Settings - Presets

Create your own UI style using [style settings](https://github.com/mgmeyers/obsidian-style-settings) plugin.

### Components

```
{
  "Components@@status-bar-autohide": true,
  "Components@@outline-enhanced": true,
  "Components@@new-tab-image-select": "new-tab-image-default"
}
```


### Editor

```
{
  "Editor@@line-hover-indicator": true,
  "Editor@@focus-indicator-codeblock-line-number": true,
  "Editor@@h2-color-select": "h2-color-default",
  "Editor@@callout-style-select": "callout-style-1"
}
```
