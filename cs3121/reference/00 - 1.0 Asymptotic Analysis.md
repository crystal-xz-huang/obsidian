## 1.1 Asymptotic Notations: O(), â„¦(), and Î˜()

> [!summary]
> - $f(n) = O(g(n))$ means "$f(n)$ grows asymptotically **slower than or equal** to $g(n)$"
> - $f(n) = \Omega(g(n))$ means "$f(n$) grows asymptotically **faster than or equal** to $g(n)$"
> - $f(n) = \Theta(g(n))$ means "f(n) grows asymptotically **equal to** $g(n)$"

The aymptotic notations big O, bigÂ Î˜ (theta), bigÂ Î© (omega) describes the limiting behavior of a function when the argument n tends towards a particular value or infinity (is large enough).

### O (big O) - upper bound

Let $f, g$ be two positive functions.  
We say that $f \in O(g)$ (or $f(n) = O(g(n))$) if there exist positive constants $c, n_0 > 0$ such that  

$$
f(n) \leq c \cdot g(n), \quad \text{for all } n \geq n_0.
$$

This characterises the notion of a *upper bound* for a function to within a constant factor, meaning the value of $f(n)$ always lies on or below $c \cdot g(n)$ for large enough $n$.

### Î© (big omega) - lower bound

Let $f, g$ be two positive functions.  
We say that $f \in \Omega(g)$ (or $f(n) = \Omega(g(n))$) if there exist positive constants $c, n_0 > 0$ such that  

$$
f(n) \geq c \cdot g(n), \quad \text{for all } n \geq n_0.
$$

This characterises the notion of a *lower bound*.

### Î˜ (big theta) - tight bound

Let $f$ and $g$ be two positive functions.  
We say that $f \in \Theta(g)$ (or $f(n) = \Theta(g(n))$) if $f(n) = O(g(n))$ and $f(n) = \Omega(g(n))$.  

This characterises the notion of a *tight bound*.

### ðŸš€ Growth Rate Hierarchy

From <u>slowest â†’ fastest</u> as $n \to \infty$:

| Category          | Functions                                  | Examples           |
| ----------------- | ------------------------------------------ | ------------------ |
| Constant          | constant values                            |                    |
| Logarithmic       | $\log(\log n) < \log n < (\log n)^k$       |                    |
| Polynomial Roots  | $n^{1/100} < n^{1/2} = \sqrt n$            |                    |
| Polynomial        | $n < n \log n < n^{1+\epsilon} < n^k$      | $n^{1.001}< n^{2}$ |
| Exponential       | $a^n, \; a>1$                              | $2^n < 3^n < e^n$  |
| Super-Exponential | $n! < n^n$  (factorials, powers of powers) | n                  |

$$1< \log(\log n) \;<\; \log n \;<\; (\log n)^k \;<\; n \;<\; n \log n \;<\; n^{1+\epsilon} \;<\; n^c \;<\; a^n \; (a > 1)\;<\; n! \;<\; n^n$$

- **Constants** don't grow at all.
- **Logs** grow slowly towards infinity.
- **Polynomials** grow faster the higher their degree. 
- **Factorials** beat exponentials with a constant base
- **Exponentials** grow faster the larger their base.
