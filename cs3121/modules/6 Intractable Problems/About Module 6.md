# Module 6: Intractable Problems


## Module Learning Outcomes

- Identify when a problem is intractable  
- Explain relationships between intractable problems
- Use these relationships to obtain techniques for solving intractable problems, and to understand which running times are achievable.
- Solve intractable problems exactly and approximately using a variety of algorithm design techniques
