## Keyboard Shortcuts

- ⌥: Option key (Alt on Windows)
- ⌃: Control key 
- ⇧: Shift key

The Control (Ctrl) key is used for "right-clicking" (secondary click) and for text-editing shortcuts
The Command key (modifier key) is used for keyboard shortcuts.

- **⌘ + ⌥ + Click** (CMD + OPT + Click) or **Right-Click > Open to the right** 



| Keybinding            |                 | Command                  |
| --------------------- | --------------- | ------------------------ |
| ⌘P                    | Cmd + P         | Open Command Palette     |
| ⌘;                    | Cmd + `;`       | Open Settings            |
| ⇧⌘P                   | Shift + Cmd + T | Open Templater           |
| ==Rich editing==      |                 |                          |
| ⇧⌥F                   | Shift + Opt + F | Lint current file        |
| ⇧⌘V                   | Shift + Cmd + V | Paste as plain text      |
| ==Text selection==    |                 |                          |
| ⌥⌘\                   | Opt + Cmd + \   | Clear formatting         |
| ⌥⌘0                   | Opt + Cmd + 1   | Toggle heading           |
| ⌥⌘1 / ⌥⌘2 / … / ⌥⌘6 |                 | Set heading 1, 2, …, 4 |
| ⌥⌘.                   | Opt + Cmd + `.` | Toggle blockquote        |

Display

| Keybinding |                       |                                 |
| ---------- | --------------------- | ------------------------------- |
| ⇧⌘E        | Shift + Cmd + E       | Toggle Source/Live Preview mode |
| ⌘] / ⌘\[   | Cmd + `[` / Cmd + `]` | Toggle left/right sidebar       |
| ⌘0         | Cmd + 0               | Reset zoom                      |
